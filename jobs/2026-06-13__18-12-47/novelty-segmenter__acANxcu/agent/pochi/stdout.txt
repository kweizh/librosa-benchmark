## What the script does

| Step | Detail |
|---|---|
| **Feature extraction** | 20 MFCCs + Δ + ΔΔ stacked into a 60-dim feature matrix (HOP=512) — L2-normalised per frame |
| **SSM** | `librosa.segment.recurrence_matrix(mode='affinity', metric='cosine', full=True)` → 646×646 affinity matrix |
| **Checkerboard kernel** | Hand-built 128×128 matrix: +1 in top-left / bottom-right quadrants, −1 in the off-diagonal quadrants, tapered with a 2-D Gaussian window (σ = half-size / 2) |
| **Novelty curve** | Frame-by-frame Frobenius inner product of the kernel with the diagonal sub-matrix of the padded SSM, then lightly smoothed (Gaussian σ=2) |
| **Peak-picking** | `scipy.signal.find_peaks` with height = mean + 0.75σ of the interior novelty, min distance ≈ 1.5 s; margin zone (1.5 × kernel half-size from each edge) is excluded |
| **Time conversion** | `librosa.frames_to_time` with the same sr / hop_length |
