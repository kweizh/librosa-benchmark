### Implementation Details:
1. **Audio Loading**: Preserves native sample rate and multi-channel configurations by using `librosa.load(..., sr=None, mono=False)`.
2. **Global RMS Calculation**: Computes the exact RMS across all channels and samples using `np.sqrt(np.mean(y**2))`.
3. **Loudness Normalization**:
   - The default target is `-20 dBFS` (linear amplitude `0.1`).
   - Clipping is prevented by checking if applying the target gain would cause the peak absolute sample amplitude to exceed `1.0`.
   - If clipping would occur, the maximum safe gain `1.0 / peak_orig` is applied.
4. **Audio Saving**: Writes the normalized audio back to `/workspace/outputs/` using `soundfile.write`. It correctly transposes multi-channel signals since `librosa` returns channels-first while `soundfile` expects channels-last.
5. **JSON Report**: Generates `/workspace/report.json` with keys `orig_rms_db`, `gain_db`, `final_rms_db`, and `peak_after` for each processed file. All values are cast to Python float types for safe JSON serialization.
