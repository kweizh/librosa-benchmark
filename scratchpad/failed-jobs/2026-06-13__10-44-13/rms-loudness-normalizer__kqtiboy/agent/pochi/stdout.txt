Created `/workspace/normalize.py` — a batch RMS loudness normalizer that processes all WAV files under `/workspace/inputs/` and writes normalized outputs to `/workspace/outputs/` plus a JSON report to `/workspace/report.json`.

**How it works:**
1. Loads each WAV with `librosa.load(sr=None, mono=False)` to preserve native sample rate and channel layout
2. Computes global RMS across all channels/samples: `√(mean(y²))`
3. Calculates the gain needed to reach the -20 dBFS RMS target (linear amplitude 0.1)
4. If that gain would push peak above 1.0 (clipping), reduces gain to `1.0 / peak` instead
5. Applies the gain and writes the output with `soundfile.write`, transposing multi-channel data from channels-first to channels-last
6. Records `orig_rms_db`, `gain_db`, `final_rms_db`, and `peak_after` per file
